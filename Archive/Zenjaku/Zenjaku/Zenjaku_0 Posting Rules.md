# **ZENJAKU_0 — POSTING RULES (AUTOMATED ACCOUNT)**

Only post information that would still be true if no one was watching.

Facts only
No intention
No explanation

No future tense

## **Forbidden Content**

Do NOT post:

- “We / I” language    
- Explanations
- Intentions
- Roadmaps
- “Coming soon”
- Emotional tone
- Emojis
- Replies    
- Threads
- Calls to action
