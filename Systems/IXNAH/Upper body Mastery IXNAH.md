# IXNAH — Upper Body Mastery Logic (Final, Minimal, Efficient)

## Core Principle
**Mastery → Load → Mastery again.**

I do not chase new exercises.  
I deepen the same ones until they break me open — then I add weight and repeat.

This guarantees:
- functional muscle
- visible presence
- total-body control
- zero wasted movement
- high aura / minimal system

---

## Definition of Mastery (Non-Negotiable)

A movement is **mastered** only when it is owned in **all three domains**:

### 1. Range
- Full, deepest safe ROM  
- No joint collapse  
- No shortcuts  

### 2. Load
- External weight added when bodyweight is no longer limiting  
- Strength increases without loss of mobility  

### 3. Expression
- **Slow** (control, tension)
- **Explosive** (rate of force)
- **Isometric** (calm under load)

If one domain is missing → it is not mastered.

## Progression Law (Simple, Brutal)

When a movement feels “easy”:

1. Slow it down  
2. Add pauses  
3. Add range  
4. **Add weight**  
5. Repeat mastery cycle  

I do **not** add new movements.  
I increase **depth**, not breadth.
