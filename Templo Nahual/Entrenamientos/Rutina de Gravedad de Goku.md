El entrenamiento de gravedad es un entrenamiento dodne se combina el dominio de la energía, uso del ki, calistenia con peso, y enfocado a ganar fuerza y movilidad, no hipertrofia.

Se caracteriza por hacer calistenia, esnfocada de ganar fuerza
Calistenia con skills, para ganar control interno
Uso de Qi Gong para cultivar el qi
Uso de chaleco con peso, para generar fuerza y potencial de explosividad
Ejercicios básicos, pero con progresiones, para lograr skills avanzadas (ej. handstand pushups)

En este entrenamiento no se entrena explosividad ni golpes, pues es el entrenamiento de gravedad.
