# THE NAHUAL CREATION RULE

1. TRAIN DAILY.
   - Body, spirit, mind. No exceptions.

2. CREATE FOR FUN ONLY.
   - If it’s fun, it stays.
   - If it’s not fun, it gets dropped instantly.

3. PUBLISH FAST.
   - No perfection.
   - No polishing hell.
   - If it's done enough to enjoy, it gets released.

4. MOVE TO THE NEXT CREATION.
   - No attachment.
   - No overthinking.
   - No “should I tweak it?”

5. DO NOT MODIFY THIS RULE.
   - Once set, it stays.
   - No adding extra layers.
   - No new “systems.”
   - Execute the formula exactly as written.
