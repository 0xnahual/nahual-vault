# Game Design Document: Nahual Archetypes System

## Core Concept
- The world is built on the **9 Pillars of Nahualism**, each representing a path of power.  
- In **Multiplayer (PvP)**, players embody **Archetypes** — Hearthstone-style classes anchored to one Pillar.  
- In **Single Player (Roguelike)**, the player walks the path of the **Nahual** — mastering all pillars through trials, unlocking hybrid superpowers and overlapping martial arts expressions.

## Overlap Martial Arts (Shared Techniques)

Core moves interpreted differently by each Pillar.  

- **Ma Bu (Horse Stance)**  
  - Voluntad: inner flame → explosive dash.  
  - Balance: rooted stance → seismic shockwave.  
  - Energía: channel qi → stamina recovery.  
  - Presencia: anchor → slows enemy time.  

- **Clinch / Grapple**  
  - Voluntad: burning clinch, flame damage over time.  
  - Balance: earth-rooted slam.  
  - Fluir: throw redirecting momentum like a river.  
  - Entrega: jaguar spirit claws emerge during grapple.  

- **Strike (Straight Punch / Kick)**  
  - Atención: precise, piercing → crits on accuracy.  
  - Intención: guided → auto-corrects trajectory.  
  - Voluntad: fiery → scales with missing HP.  
  - Desapego: airy → phased hit ignores armor.

## Player Experience

- **PvP (Hearthstone Style):**  
  - Clean archetypes tied to one pillar.  
  - Rare overlap superpowers = ultimates/ritual cards.  

- **Single Player (Roguelike):**  
  - Player unlocks and stacks powers from multiple pillars.  
  - Martial arts overlaps evolve into god-tier Nahual moves.  
  - Endgame fantasy = become the **Nahual** who masters all 9.  

---

## Design Pillars for System
- PvP = **clarity + balance** (Hearthstone).  
- Single Player = **freedom + mastery** (roguelike).  
- Martial Arts = **bridge mechanic** across pillars, creating overlap superpowers.  
- Archetype names = clarity, Pillar names = lore.
