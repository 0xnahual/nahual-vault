## SECTION 7: DATABASE STRUCTURE REFERENCE

### Data Source IDs (Notion collection IDs)
- **Daily Macros:** `collection://1f980f06-c78d-80fa-a0c0-000b9cf77ac3`
- **Food Log:** `collection://1f980f06-c78d-80c8-aeff-000b17009a4c`
- **Calorias (food reference):** `collection://1f780f06-c78d-80c5-a8b1-000bd79b324b`
- **Supplements:** `collection://1f780f06-c78d-807b-b7df-000b16712112`

### Database Relationships
```
Food Log Entry
  |-- "Comida" relation --> Calorias (food item)
  |-- "Day" relation --> Daily Macros (day entry)
  |-- "Porciones" (number) = servings count
  |-- "Tiempo" (select) = Desayuno | Comida | Cena | Snack | Pre-Train
  |-- Formula fields: Calorias (kcal), Proteinas (gramos), Carbohidratos (gramos), Grasas (gramos)
  |     (calculated as: Calorias * Porciones from linked Comida item)

Daily Macros
  |-- "Fecha" (date)
  |-- Rollup fields: Calorias Totales, Proteinas Totales, Carbohidratos Totales, Grasas Totales
  |     (sum of all linked Food Log entries' formula values)
  |-- "Notes" (rich text)

Calorias (Food Reference)
  |-- "Name" (title) = food item name
  |-- "Calorias" (number) = kcal per portion
  |-- "Proteina" (number) = grams per portion
  |-- "Carbohidratos" (number) = grams per portion
  |-- "Grasa" (number) = grams per portion
  |-- "Tipo de Porcion" (rich text) = portion description

Supplements
  |-- "Name" (title) = "Supplements - [date]"
  |-- "Date" (date)
  |-- "Status" (select) = Today | Done
  |-- 8 checkbox properties: Colageno, Creatina, Dabex XR, Magnesio, Melena de Leon, Omega 3, Potasio, Z Blend
```

### API Limitations Encountered
- Rollup fields return `<omitted />` -- cannot read computed daily totals
- Formula fields return `formulaResult://` references -- cannot read computed per-entry macros
- Search API returns max 10 results per query -- cannot enumerate all food log entries
- Workaround: resolve Comida relation manually and multiply by Porciones
