La ecualización dinámica sirve para sacar brillo u obscuridad a ciertos sonidos, dependiendo de otros sonidos. Es cómo hacer sidechain con el brillo.

A diferencia de el sidechain, la ecualización dinámica sube o baja de volumen ciertas frecuencias que sean necesarias, mientras que el sidechain lo hace con todo el sonido.

## Diferencias entre sidechain y EQ dinámica.

| Aspecto              | Ecualización Dinámica                                                             | Sidechain                                                                            |
|----------------------|-----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------|
| **Función Principal**| Ajustar dinámicamente las ganancias de frecuencias específicas dentro de una misma señal. | Controlar la dinámica de una señal utilizando la información de otra señal.          |
| **Control**          | Basado en la amplitud de las frecuencias dentro de la misma pista.                | Basado en la señal de una pista diferente.                                           |
| **Aplicaciones**     | Corrección y modelado tonal flexible dentro de una pista.                         | Creación de espacio, efectos rítmicos y control de interacción entre pistas.         |
| **Ejemplo Común**    | Reducir automáticamente una frecuencia nasal en una voz solo cuando es necesario. | Hacer que el bajo "retroceda" momentáneamente cada vez que el bombo golpea.           |
